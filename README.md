## Features

- **Native** — Written in pure Swift and truly native to embrace the Mac experience
- **Free to download** — Free to try. Upgrade for unlimited bookmarks and advanced features
- **No tracking** — No analysis and no data collecting
- **Launch at login** — Always available when your Mac awakes
- **Dark mode** — Gorgeous light and dark mode themes
- **Multiple content types** — Supports files, folders, links, text and images
- **Keyboard shortcut** — Quick access without lifting your hands off the keyboard

<br/>

## Easy access from menu bar

Drag any file, link or text directly into the Coolbar menu bar icon to save it as a bookmark. Access everything with a single click.
